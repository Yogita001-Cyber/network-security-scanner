# 🛡️ Network Security Scanner \& Reporting Tool

A professional Python-based cybersecurity solution for automated network scanning, service detection, risk analysis, and security report generation.

\---

## 📌 Overview

The **Network Security Scanner \& Reporting Tool** is designed to help security analysts, students, and cybersecurity enthusiasts perform efficient defensive network assessments.

This tool automates the process of network discovery, port scanning, service identification, and risk categorization, while generating clean and professional reports for analysis and documentation.

It provides a lightweight yet scalable approach to understanding network exposure and identifying potentially vulnerable services within a target environment.

\---

## 🚀 Key Features

* 🔍 Automated network and host discovery
* 🌐 Multi-target scanning support
* 🔓 Open port and service detection using Nmap
* ⚠️ Intelligent risk classification  
*(High / Medium / Low / Unknown)*
* 📊 Professional dashboard-style PDF reports
* 📁 Multiple export formats:

  * TXT
  * JSON
  * PDF
* 🧠 Structured findings with security recommendations
* ⚡ Fast and lightweight scanning workflow

\---

## 🛠️ Technologies Used

* **Python** — Core application development
* **Nmap** — Network scanning and service enumeration
* **ReportLab** — PDF report generation
* **JSON** — Structured data storage and export

## 📋 Prerequisites

Before running the project, ensure the following tools are installed:

* Python 3.10 or later
* Nmap

Verify Nmap installation:

```bash
nmap --version
```

\---

## ⚙️ Installation

Clone the repository and install dependencies:

```bash
git clone (https://github.com/Yogita001-Cyber/network-security-scanner)

cd network-security-scanner

pip install -r requirements.txt
```

\---

## ⚡ Quick Start

```bash
# Install dependencies
pip install -r requirements.txt

# Verify Nmap installation
nmap --version

# Launch the scanner
python main.py
```
